WELCOME!
--------
Thank you for playing Othello! This is a feature-rich implementation
of the classic strategy board game, also known as Reversi.


QUICK START
-----------
1. Double-click Othello.exe to launch the game
2. Click "PLAY" from the main menu
3. Choose your game mode:
   - Play vs AI: Challenge computer opponents
   - Play Friend: Local 2-player mode
4. Select difficulty (if playing AI):
   - Easy: Good for beginners
   - Medium: Balanced challenge
   - Hard: For experienced players
5. Click on green circles to make your move
6. Flip opponent's discs by sandwiching them
7. Have fun!


GAME RULES
----------
• Players take turns placing discs on the 8×8 board
• Place your disc to sandwich opponent's discs
• Sandwiched discs flip to your color
• Valid moves are shown with green pulsing circles
• Last move is highlighted with a yellow glow
• If you have no valid moves, your turn is skipped
• Game ends when no player can move
• Player with the most discs wins!

HOW TO PLAY
-----------
PLACING DISCS:
  • Click on any green circle to place your disc
  • Your disc must flip at least one opponent disc
  • Flips occur in all 8 directions (horizontal, vertical, diagonal)

STRATEGY TIPS:
  • Control the corners - they can't be flipped!
  • Edges are strong positions
  • Don't give opponent corner access
  • Mobility is key - keep your options open
  • The game can turn quickly near the end

CONTROLS:
  • Mouse: Click to select moves and buttons
  • Settings (⚙): Access in-game options
  • ESC/Close: Exit game (with confirmation)


SETTINGS PANEL

  RESTART      - Start a new game

TROUBLESHOOTING
---------------
Game won't start:
  → Run as administrator
  → Check Windows Defender isn't blocking
  → Install Visual C++ Redistributables

Slow performance:
  → Close other applications
  → Update graphics drivers
  → Run in windowed mode

No sound:
  → Check Windows volume mixer
  → Verify audio output device is working
  → Restart the game

Visual glitches:
  → Update graphics drivers
  → Try compatibility mode (Windows 7/8)

FREQUENTLY ASKED QUESTIONS
--------------------------
Q: Can I undo a move?
A: Not in current version. Think carefully before placing!

Q: How do I win?
A: Have the most discs when the game ends (no more valid moves).

Q: Can AI make mistakes?
A: Yes, especially on Easy difficulty. Hard is very challenging!

Q: What happens if I can't move?
A: Your turn is skipped automatically. Game ends if both can't move.

Q: Are the corners really important?
A: Yes! Corners can never be flipped, making them the strongest positions.

Q: Can I play online?
A: Network multiplayer is available (requires manual setup).
